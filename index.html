Secux Linux

A security-first distribution for those who don't even trust the hardware. Out-of-the-box encryption, trusted boot, protection against TPM attacks, and application isolation and more.

Download ISO Source Code

Security System

/etc/secux/security.conf

LUKS2 by Default

Full Disk Encryption (FDE). Argon2id protects against password brute-forcing better than classic methods.

Zero Trust

A continuous chain of trust from UEFI to system login. Runtime attestation (SIRA) without flexibility loss.

IDP Protection

An additional layer of cryptographic protection on top of TPM. Protects keys from physical attacks and voltage manipulation.

Hardened Kernel

Kernel choice during installation: Hardened (AVX optimizations, CFI, memory protection) for the paranoid, LTS for stability, or plain Linux.

Sandboxing

Applications from the Security Manager are distributed as Flatpaks with reinforced system isolation.

Stability

You can use exclusive in-house software repositories, independent of Arch Linux.

Installation Options
