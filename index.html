СКАЧАТЬ ИСХОДНЫЙ КОД

СИСТЕМА ЗАЩИТЫ

/etc/secux/security.conf

[ DISK_ENCRYPTION ]

LUKS2 ПО УМОЛЧАНИЮ

Полное шифрование диска (FDE) сразу после установки. Современный стандарт LUKS2 с алгоритмом Argon2id для защиты от перебора паролей.

[ DEFENSE_IN_DEPTH ]

IDP

Дополнительный слой криптографической защиты поверх TPM. IDP защищает ключи шифрования даже от физических атак с манипуляцией напряжением.

[ TRUSTED_BOOT ]

ДОВЕРЕННАЯ ЗАГРУЗКА

Полностью проверяемая цепочка доверия от UEFI до экрана входа. Гарантирует, что вредоносный код не был внедрен и ваши данные в безопасности.

[ KERNEL_SPACE ]

На вкус и цвет ядра разные

Выбирайте при установке:
1. linux-hardened - максимальная безопасность.
2. linux-lts - стабильность.
3. linux - новейшее оборудование.

[ ISOLATION ]

Песочница

Дополнительная изоляция. Flatpak пакеты из Security Manager поставляются с усиленной изоляцией: никакого доступа к домашней папке.

[ ROLLING_RELEASE ]

БАЗА ARCH LINUX

Вы получаете обновления безопасности через часы после их выхода, а не через недели.

ВАРИАНТЫ УСТАНОВКИ

Системные требования

[ МИНИМАЛЬНЫЕ ]

Чтобы кое-как запустилось
  • ОЗУ 1 ГБ
  • Процессор Да
  • Диск 15 ГБ
  • Прошивка UEFI

[ РЕКОМЕНДОВАННЫЕ ]

Для комфортной работы
  • ОЗУ 4 ГБ
  • Процессор Многоядерный
  • Диск 30 ГБ SSD
  • Безопасность TPM 2.0
  • Прошивка UEFI + SB Keys
    ( Возможность замены ключей Secure Boot )
