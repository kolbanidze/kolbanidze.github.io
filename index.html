Система защиты

/etc/secux/security.conf

LUKS2 по умолчанию

Полное шифрование диска (FDE). Argon2id защищает от перебора паролей лучше классических методов.

Доверенная загрузка

Непрерывная цепочка доверия от UEFI до входа в систему. Гарантия целостности загрузчика.

IDP Protection

Дополнительный слой криптографической защиты поверх TPM. Защита ключей от физических атак и манипуляций напряжением.

Hardened Kernel

Выбор ядра при установке: Hardened для параноиков, LTS для стабильности или Zen для скорости.

Песочница

Приложения из Security Manager поставляются в Flatpak с усиленной изоляцией от системы.

Rolling Release

База Arch Linux. Обновления безопасности приходят через часы после обнаружения уязвимости.
